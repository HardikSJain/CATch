## 💡 Core Features

### 1. Daily Practice System (2-Tier)
**Tier 1 - Daily Minimum (Breadth):**
- 3 DILR + 5 QA + 3 VARC (11 questions)
- Maintains all sections daily

**Tier 2 - Focused Deep Practice (Depth):**
- 30 questions in today's focus section
- Mon/Thu: Quants
- Tue/Fri: VARC  
- Wed/Sat: DILR
- Sun: Mock/Review

### 2. Progress OS
- **Calendar View:** Color-coded days (🟢 complete | 🟡 partial | 🔴 incomplete)
- **Streak Counter:** 🔥 Gamification for consistency
- **Stats Dashboard:** Section-wise, topic-wise breakdown
- **Weak Area Heatmap:** Know exactly where to improve

### 3. Question Bank
- 100+ questions (MVP) → 1000+ (by CAT day)
- From Arun Sharma (12th Edition)
- With images (diagrams, charts, caselets)
- MCQ + TITA support
- Instant explanations

### 4. Smart Scheduling
- Auto-assigns focus topics
- Customizable weekly rotation
- Daily targets (adjustable)
- Mock mode (coming V2)

### 5. Offline-First
- No internet required
- All data local (SQLite)
- Fast, instant loading
- Privacy-first (no data sent anywhere)
