## Key Strategies

**DILR:**
- Set selection > Speed
- Spend 5 min scanning all sets
- Attempt 3-4 sets perfectly
- 85%+ accuracy in attempted sets

**VARC:**
- Reading speed via daily articles
- Skip 1 hard passage
- Attempt all VA questions
- 80%+ accuracy

**QA:**
- Speed via shortcuts
- Skip aggressively (if >3 min, move on)
- Prioritize TITA (no negative marking)
- 90%+ accuracy
